# Portfolio Website - Apurv Deshpande

This portfolio website showcases my professional experience, projects, and personal background . It also has distinct pages for some of my projects in software development, data engineering, and more. Feel free to explore the different sections (experience, skills, projects) and get in touch!

## Deployment: 
This portfolio is deployed on Vercel: https://portfolio-orpin-six-86.vercel.app/

## Developer setup (Vite + React + Tailwind)
To run the project locally after this scaffold:

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev / npm start
